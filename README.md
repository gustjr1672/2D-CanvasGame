## 게임소개 🕹️

### INTO THE TROPHY 🏆
숨겨진 비밀의 해답을 찾아서 캐릭터를 트로피 속으로 넣어보세요 <br>

#### 이용 방법
원하는 캐릭터를 선택하고, 방향키로 캐릭터를 움직여보세요.

### 게임 구성

🥇 총 6개의 스테이지를 차례대로 도전해보세요.

🥈 이전 단계를 클리어하면, 다음 단계에 도전할 수 있어요!!
